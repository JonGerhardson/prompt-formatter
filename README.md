# prompt-formatter
Easy GUI for formatting long context length meeting summarizer prompt 

Lets you paste text or load a txt or md file which gets added to a specific prompt I am using to generate reports from meeting transcripts. Very minor conveieince that I used an LLM to generate, idea to github repo in like 5 minutes. 

usage

```
python prompt_formatter.py
```
